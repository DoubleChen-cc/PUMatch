# Fig9 One-Click Pipeline

This directory provides a complete pipeline:  
run `PUMatch`, `GAMMA`, and `STMatch` -> aggregate results -> generate figures with `plot_fig9.py`.

---

## Files

- `run_fig9_pipeline.py`  
  Main one-click script (cross-platform, recommended)
- `run_fig9_pipeline.sh`  
  Linux/macOS wrapper (internally calls the Python script above)
- `prepare_fig9_input.py`  
  Aggregates three `result.txt` files into `result.xlsx` (`Sheet1`)
- `plot_fig9.py`  
  Reads `result.xlsx` and generates Fig9 PDFs

---

## Requirements

- Python 3.8+
- Python packages:
  - `pandas`
  - `numpy`
  - `matplotlib`
  - `openpyxl`

Install example:

```bash
pip install pandas numpy matplotlib openpyxl
```

---

## Prerequisites

Make sure the following executables are built and runnable (default paths):

- `./test` (project root)
- `./compared_systems/GAMMA/sm`
- `./compared_systems/STMatch_UM/cu_test`

Also make sure query graphs are in `pattern/*.g` (all are used by default).

---

## One-Click Run

Run from the project root:

```bash
python3 plot/run_fig9_pipeline.py --graph <data_graph_path> --clear-result
```

Or use the shell wrapper:

```bash
bash plot/run_fig9_pipeline.sh --graph <data_graph_path> --clear-result
```

Example:

```bash
python3 plot/run_fig9_pipeline.py --graph /data/friendster
```

---

## Common Arguments

- `--patterns "pattern/1.g,pattern/2.g"`  
  Run only specific query graphs; default is all `pattern/*.g`
- `--clear-result`  
  Clear the three `result.txt` files before running
- `--skip-run`  
  Skip program execution and only do aggregation + plotting
- `--pumatch-bin` / `--gamma-bin` / `--stmatch-bin`  
  Custom executable paths
- `--pumatch-extra-args` / `--stmatch-extra-args`  
  Pass through extra arguments
- `--gamma-mt` (default: `1`)
- `--gamma-mode` (`full` or `truncate10`, default: `full`)
- `--gamma-debug-flag` (default: `debug`)

---

## Outputs

- Aggregated Excel: `plot/result.xlsx` (`Sheet1`)
- Figure files: generated by `plot_fig9.py` in the `plot/` directory (multiple PDFs)

---

## Re-Plot Only (Without Re-Running Programs)

```bash
python3 plot/run_fig9_pipeline.py --graph <data_graph_path> --skip-run
```

This directly reads existing `result.txt` files, rebuilds the Excel input, and re-generates figures.
